# MyServer – Hosting Plans Page
![Status](https://img.shields.io/badge/Status-Concluído-brightgreen)
![License](https://img.shields.io/badge/Licença-MIT-blue)
![Tech](https://img.shields.io/badge/Feito%20com-HTML%20%26%20CSS-orange)

## Overview
MyServer is a static webpage created to present hosting plans in a clear, modern, and responsive way.
The goal is to simulate a commercial landing page, showcasing three pricing plans with highlighted features and visual emphasis.

Built using pure `HTML` and `CSS`, focusing on structure, semantic markup, and responsiveness.

Live version:
https://lucascarvalho-oliveira.github.io/planos.html

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- Flexbox
- Responsive Design (Media Queries)

## Installation
1. Clone the repository:
git clone https://github.com/lucascarvalho-oliveira/planos.html.git
2. Open the HTML file in your browser:
- Locally: double-click planos.html
- Or access the online version via GitHub Pages

## Features
- Display of three hosting plans: **Essential**, **Premium**, and **Gold**
- Each plan includes:
  - Monthly price
  - Available features
  - Subscription button
- Fully responsive layout (desktop → mobile)
- Navigation bar with active section highlight
- Dark color palette with strong visual contrast
